Habilidades

C/C++

C++ é uma linguagem estruturada com orientação a objeto muito popular comercialmente. Possuo domínio amplo da linguagem e capacidade de implementação de novos recursos/bibliotecas/estruturas de dados (como no projeto de TADs).

LaTeX

LaTeX é uma uma linguagem de diagramação de texto muito utilizada em textos cientifícos e matemáticos que proporciona diversas vantagens, como textos formatados automaticamente (inclusive com regras da ABNT), referências cruzadas, etc. Possuo domínio sobre os funcionamentos padrões do LaTeX, além de conhecimento extra para customização e adptação de documentos a outros estilos (você pode conferir no projeto BTI Documents).
